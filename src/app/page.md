## このドキュメントについて

TypeScript、React、Vite、React Router、Tailwind CSS、TanStack Query を用いて「ポケモン図鑑アプリ」を完成させるカリキュラムです。各章は短いステップで進み、重要ポイントにはコードと背景知識を併記します。

## 学習ロードマップ

- セットアップとプロジェクト作成（/docs/section-03, /docs/05-vite）
- ルーティングとページ構成（/docs/07-react-router）
- スタイルとデザイン（/docs/08-tailwind-css-config）
- データ取得とキャッシュ（/docs/09-pokeapi, /docs/10-tanstack-query）

## 必要環境

Node.js / npm、Git、ブラウザ（Chrome/Edge/Safari）。VS Code 推奨。
