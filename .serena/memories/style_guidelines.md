# Style Guidelines
- Use TypeScript in the Next.js app router structure under `src/app`.
- Formatting handled by Prettier with `singleQuote: true` and `semi: false`; Tailwind classes sorted via `prettier-plugin-tailwindcss`.
- Favor functional React components and Next.js conventions; co-locate component styles in Tailwind utility classes.
- Follow conventional component naming (`PascalCase`) and file naming in kebab-case for routes; Markdown/Markdoc files live under `src/markdoc`.
